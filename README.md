# 웨어러블 기기를 이용한 코로나19 감염 예측 어플리케이션

> **관련 레포지토리 (LSTM 예측 모델):** https://github.com/j-sindol/covidapp-wearable.git

---

## 프로젝트 개요

코로나19 자가진단 키트의 낮은 정확도와 즉각적인 결과 도출의 한계를 보완하기 위해, **웨어러블 기기의 생체 데이터(심박수·혈중산소농도)** 와 **증상 설문조사**를 결합하여 코로나19 감염 위험도를 예측하는 안드로이드 어플리케이션입니다.

## 머신러닝 모델

### 1. LSTM 예측 모델 (웨어러블 데이터)

- **입력**: 심박수(HR) + 혈중산소농도(SpO2) 120분 시계열 데이터
- **출력**: 다음 심박수 및 혈중산소농도 예측값
- **전처리**: MinMax 스케일링 적용 (HR: 40–180, SpO2: 70–100)
- **변환**: TensorFlow Lite(`HRSpO2model.tflite`)로 변환하여 앱 내 탑재

### 2. Logistic Regression 모델 (증상 설문)

- **데이터셋**: `covid-19 symptoms dataset.csv`
- **분할**: Train 70% / Test 30%
- **활용**: 각 증상 변수의 coefficient를 추출하여 위험도 점수 계산에 사용

| 증상 변수 | Coefficient |
|-----------|-------------|
| Runny Nose (콧물) | +0.0984 |
| Sore Throat (인후통) | +0.0640 |
| Tiredness (피로감) | +0.0546 |
| Difficulty Breathing (호흡 곤란) | -0.0718 |
| Body Pain (몸살) | -0.0928 |
| Nasal Congestion (코막힘) | -0.0328 |
| Diarrhea (설사) | -0.0075 |
| Dry Cough (기침) | -0.0003 |

## 어플리케이션 화면 구성

### 1. 시작 화면
앱 런처 화면.

### 2. 로그인 화면
사용자 인증 후 메인 화면으로 이동.

### 3. 국내 코로나 현황 화면 (`MainActivity`)
질병관리청 코로나19 현황 페이지를 WebView로 실시간 표시.

### 4. 웨어러블 데이터 확인 화면 (`WearableActivity`)
- 120분간 축적된 심박수·혈중산소농도 데이터를 꺾은선 그래프로 표시
- LSTM 모델로 예측된 다음 심박수 및 혈중산소농도 표시
- 건강 상태 판정

| 지표 | 기준 | 판정 |
|------|------|------|
| 심박수 | 50–75 bpm | 정상 |
| 심박수 | < 50 또는 ≥ 75 | 낮음 / 높음 |
| 혈중산소농도 | ≥ 80% | 정상 |
| 혈중산소농도 | < 80% | 낮음 / 매우 낮음 |

### 5. 설문조사 화면 (`TestActivity`)
아래 11개 항목에 예/아니오로 응답하고, 체온을 직접 입력.

| 항목 | 항목 |
|------|------|
| 질병 유무 | 콧물 |
| 호흡 곤란 | 피로감 |
| 기침 | 인후통 |
| 과거 코로나 확진 | 코막힘 |
| 설사 | 확진자 접촉 여부 |
| 기저질환 유무 | — |

### 6. 결과 확인 화면 (`ResultActivity`)
웨어러블 점수와 설문 점수를 합산하여 100점 만점 기준 위험도 표시.

---

## 위험도 점수 산출 방식

### 웨어러블 점수 (최대 20점)

| 지표 | 조건 | 점수 |
|------|------|------|
| 심박수 | 50 ≤ HR < 75 | 0 |
| | HR < 50 | 2 |
| | 75 ≤ HR < 85 | 5 |
| | 85 ≤ HR < 100 | 7 |
| | HR ≥ 100 | 10 |
| 혈중산소농도 | SpO2 ≥ 95% | 0 |
| | 90 ≤ SpO2 < 95% | 5 |
| | 75 ≤ SpO2 < 90% | 7 |
| | SpO2 < 75% | 10 |

### 설문 점수 (최대 80점)

```
설문 점수 = (각 증상 × LR coefficient 합계) × 70
           + 질병 유무 × 10
           + 기저질환 × 10
```

### 최종 위험도 판정

| 점수 범위 | 판정 | 색상 |
|-----------|------|------|
| 0 – 40 | 안전한 상태 | 파란색 |
| 41 – 60 | 주의 | 노란색 |
| 61 – 80 | 위험 | 주황색 |
| 81 – 100 | 매우 위험 | 빨간색 |

## 어플리케이션 실행 방법

1. 어플리케이션 실행 후 로그인한다.
2. 하단 메뉴바 **웨어러블 데이터** 탭 → **데이터 불러오기** 버튼 클릭  
   → 120분 웨어러블 데이터 로드 및 LSTM 모델 추론 실행, 예측값 저장
3. 하단 메뉴바 **설문조사** 탭 → 증상 설문 작성 후 저장
4. 하단 메뉴바 **결과확인** 탭 → 코로나19 감염 위험도 확인

## 프로젝트 목적

기존 코로나19 자가진단 키트는 정확도가 낮고 신체 증상·기저질환 유무를 반영하지 못하며 즉각적인 결과를 제공하지 못한다. 본 어플리케이션은 웨어러블 센서 데이터와 증상 설문을 결합하여 **더 정확하고 즉각적인 감염 위험도 예측**을 제공하는 것을 목표로 한다.
